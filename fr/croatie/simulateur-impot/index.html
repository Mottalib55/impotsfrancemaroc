Guide complet de la fiscalite croate

La Croatie possede un systeme fiscal relativement simple avec le Porez na dohodak (impot sur le revenu). Depuis le 1er janvier 2023, la Croatie est membre de la zone euro, ce qui simplifie les transactions financieres pour les expatries francais. Le systeme fiscal croate utilise un bareme progressif a deux tranches, ce qui le rend particulierement attractif pour les travailleurs a revenus moyens.

Tranches d'imposition 2026 (Porez na dohodak)

L'impot croate sur le revenu utilise un systeme a deux tranches tres simple :

  • 0 EUR - 50,400 EUR : 20% (taux normal)
  • Au-dela de 50,400 EUR : 30% (taux superieur)

Note : Un abattement personnel (osobni odbitak) de 560 EUR/mois (6,720 EUR/an) s'applique, reduisant la base imposable.

Prirez (Surtaxe municipale)

Surtaxe locale : 0% a 18%

  • Zagreb : 18% (taux maximum)
  • Split, Rijeka, Osijek : 15%
  • Autres grandes villes : 10-12%
  • Petites communes : 0-10%
  • Cette surtaxe s'applique sur le montant de l'impot sur le revenu

Cotisations sociales (~20% part employe)

Le systeme social croate est gere par deux institutions principales : HZMO (retraite) et HZZO (sante).

HZMO - Hrvatski zavod za mirovinsko osiguranje

Institut croate d'assurance retraite

  • 1er pilier (15%) : Regime de base par repartition
  • 2e pilier (5%) : Epargne obligatoire capitalisee dans des fonds de pension
  • Age de la retraite : 65 ans (en augmentation progressive)
  • Minimum de cotisation : 15 ans pour une pension minimale
  • Pension moyenne : environ 500-600 EUR/mois

HZZO - Hrvatski zavod za zdravstveno osiguranje

Institut croate d'assurance maladie

  • Couverture universelle : Soins de base gratuits pour tous les assures
  • Participation aux frais : Ticket moderateur variable selon les soins
  • Assurance complementaire : Environ 10-15 EUR/mois (HZZO ou privee)
  • Medicaments : Liste positive (rembourses) et liste negative
  • Qualite des soins : Bonne dans les grandes villes, variable ailleurs

Avantages fiscaux et deductions

Osobni odbitak (Abattement personnel)

  • Montant de base : 560 EUR/mois (6,720 EUR/an)
  • Coefficient enfants : +0.7 pour le 1er enfant, +1.0 pour le 2e, +1.4 pour le 3e, etc.
  • Personnes a charge : +0.7 par membre du foyer dependant
  • Invalidite : +0.4 a +1.5 selon le degre d'invalidite

Autres deductions

  • Assurance vie : Deductible jusqu'a 2,650 EUR/an
  • Epargne retraite volontaire : Deductible jusqu'a 700 EUR/an
  • Dons : Deductibles jusqu'a 2% du revenu annuel
  • Frais de sante non rembourses : Deductibles sous conditions

Declaration fiscale (Godisnja porezna prijava)

  • Date limite : 28 fevrier de l'annee suivante
  • Obligatoire si : revenus multiples, revenus de l'etranger, plus-values, etc.
  • Systeme : e-Porezna (portail fiscal en ligne)
  • OIB : Numero d'identification personnel obligatoire (Osobni identifikacijski broj)
  • Delai de remboursement : 2 a 3 mois apres la declaration

Regime des expatries et nomades numeriques

Visa Nomade Numerique Croate

  • Duree : Jusqu'a 1 an (renouvelable)
  • Revenu minimum requis : 2,540 EUR/mois
  • Avantage fiscal : Pas d'impot croate sur les revenus etrangers pendant 1 an
  • Conditions : Travail pour une entreprise etrangere ou independant
  • Assurance : Assurance sante internationale obligatoire
