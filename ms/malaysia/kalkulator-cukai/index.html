Guide complet de la fiscalite malaisienne

La Malaisie possede un systeme fiscal territorial attractif gere par le LHDN (Lembaga Hasil Dalam Negeri), l'autorite fiscale malaisienne. Le pays applique un bareme progressif allant de 0% a 30% pour les residents fiscaux. Une caracteristique unique est que les revenus de source etrangere ne sont generalement pas imposes (sauf pour certaines exceptions depuis 2022).

LHDN - L'autorite fiscale malaisienne

Lembaga Hasil Dalam Negeri Malaysia

  • Role : Collecte des impots, administration fiscale et conformite
  • Annee fiscale : 1er janvier au 31 decembre
  • Declaration en ligne : Systeme e-Filing (ezHASiL) disponible
  • Date limite : 30 avril pour les salaries, 30 juin pour les independants
  • Resident fiscal : 182 jours ou plus de presence en Malaisie

Tranches d'imposition 2026

L'impot sur le revenu malaisien utilise un systeme de tranches progressives de 0% a 30% :

  • 0 - 5,000 MYR : 0% (exoneration de base)
  • 5,001 - 20,000 MYR : 1%
  • 20,001 - 35,000 MYR : 3%
  • 35,001 - 50,000 MYR : 6%
  • 50,001 - 70,000 MYR : 11%
  • 70,001 - 100,000 MYR : 19%
  • 100,001 - 400,000 MYR : 25%
  • 400,001 - 600,000 MYR : 26%
  • 600,001 - 2,000,000 MYR : 28%
  • Au-dela de 2,000,000 MYR : 30%

Note : Les non-residents sont imposes a un taux fixe de 30% sans tranches progressives.

EPF - Kumpulan Wang Simpanan Pekerja (KWSP)

Employees Provident Fund - Fonds de prevoyance

  • Contribution employe : 11% du salaire mensuel
  • Contribution employeur : 12% ou 13% (selon le salaire)
  • But : Epargne-retraite obligatoire
  • Retraits anticipes : Possibles pour logement, sante, education
  • Age de retrait : 55 ans pour retrait complet
  • Avantage fiscal : Contributions deductibles jusqu'a 4,000 MYR

SOCSO - Pertubuhan Keselamatan Sosial (PERKESO)

Securite sociale malaisienne

  • Contribution employe : ~0.5% du salaire
  • Contribution employeur : ~1.75% du salaire
  • Plafond : 5,000 MYR de salaire mensuel
  • Couverture : Accidents du travail, invalidite, maladies professionnelles
  • EIS (Employment Insurance) : 0.2% employe + 0.2% employeur (depuis 2018)

Deductions et abattements fiscaux

Deductions personnelles 2026

  • Deduction personnelle : 9,000 MYR automatique
  • Conjoint : 4,000 MYR (si sans revenus)
  • Enfants : 2,000 MYR par enfant (8,000 MYR si etudes superieures)
  • Parents : 1,500 MYR par parent (conditions)
  • Education/Formation : jusqu'a 7,000 MYR
  • Frais medicaux : jusqu'a 8,000 MYR
  • Style de vie : 2,500 MYR (livres, sport, internet)

Allegements speciaux

  • Assurance-vie : jusqu'a 3,000 MYR deductibles
  • EPF contributions : jusqu'a 4,000 MYR deductibles
  • Private Retirement Scheme (PRS) : jusqu'a 3,000 MYR
  • Zakat/Fitrah : 100% deductible pour les musulmans
  • Donations approuvees : jusqu'a 7% du revenu agrege

PCB - Prelevement a la source

  • PCB (Potongan Cukai Bulanan) : Equivalent du prelevement a la source
  • Calcul : Effectue par l'employeur chaque mois
  • Ajustement : Declaration annuelle pour regularisation
  • Non-residents : PCB final sans declaration necessaire
  • Remboursement : Possible si trop-percu via declaration

Avantages fiscaux pour expatries

Incitations speciales

  • Labuan : Regime fiscal avantageux pour societes offshore
  • MSC Malaysia : Incentives pour entreprises tech
  • Free Trade Zones : Exonerations fiscales specifiques
  • Pas d'impot sur les plus-values : Sauf immobilier (RPGT)
